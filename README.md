# 知识空战 Web 应用（Knowledge Air Combat）

基于 TypeScript 与现代前端工程化构建的知识空战类 Web 应用，包含完整源码（src）、构建产物（dist）、文档（docs）与构建配置。项目以交互式可视化呈现知识对抗 / 空战主题内容。

## 功能特性

- TypeScript 工程化前端
- 构建打包与本地预览（dist / index.html）
- 脚本化构建与资源处理（scripts）
- 配套文档与设计说明（docs）

## 技术栈

TypeScript / JavaScript；前端构建链（package.json / tsconfig / scripts）；模块化源码结构（src）。
